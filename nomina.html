Sistema de control de nomina

El sistema Administrar empleados, control de asistencia, control de nomina segun las deducciones de ley de Nicaragua INSS - IR - INATEC

Tecnologias utilizadas

  • Php
  • MYSQL
  • XAMPP
  • Recomendacion Se recomienda correr en xampp

Realizado en 2022


Un sistema de control de nómina con PHP y MySQL es una aplicación que permite gestionar eficientemente la información relacionada con los empleados, sus asistencias, cargos y el cálculo de nómina, teniendo en cuenta las deducciones legales aplicables en Nicaragua, como el IR (Impuesto sobre el Renta), el INSS (Instituto Nicaragüense de Seguridad Social) laboral y patronal. A continuación, se describe brevemente cómo podría funcionar este sistema:

  • Gestión de Empleados:
    • El sistema permite registrar y mantener una base de datos de todos los empleados de la empresa. Cada empleado tiene un perfil que incluye información personal, detalles de contacto y ID.
  • Control de Asistencia:

    Se lleva un registro de la asistencia de los empleados, ya sea mediante marcación de entrada/salida o el ingreso manual de horas trabajadas.

  • Cálculo de Nómina:

    Para calcular la nómina de cada empleado, se multiplican las horas trabajadas por el salario por hora correspondiente. Luego, se aplican las deducciones legales establecidas en Nicaragua. Esto incluye:

    • IR: Se calcula el IR en función del salario bruto, aplicando las tasas de impuestos vigentes según las leyes fiscales de Nicaragua.
    • INSS Laboral: Se calcula la contribución al seguro social del empleado, que es un porcentaje del salario bruto.
    • INSS Patronal: Se calcula la contribución al seguro social que la empresa debe pagar por cada empleado.
  • Seguridad y Acceso:

    Se implementa un sistema de autenticación y autorización para garantizar que solo usuarios autorizados puedan acceder y modificar la información del sistema. La contraseña de cada usuario está encriptada.
